# Asset Merger for VRChat Avatars 3.0

Avatars 3.0 になったことで競合する可能性がある以下のファイルの内容を結合 (マージ) してくれる Unity エディター拡張です。

* Animator Controller
* Expression Parameters
* Expressions Menu

なお、プロジェクト本体は GitHub にて公開されており、知識があれば誰でも無料で使用およびシステムの修正が可能です。
https://github.com/mika-f/VRChat-AssetMerger


## 注意事項

不具合などがあるかもしれません。ご了承ください。
また、結合されたファイルは想定通りに結合されているか、必ず確認および動作確認を行ってください。

なお、以下の環境にて動作を確認しております。

* Unity 2018.4.20f1
* VRCSDK3-AVATAR 2020.08.06.16.30
* VRChat 2020.3.2p1, Build 970


## GitHub 版との違いについて

BOOTH ではビルド済みの UnityPackage と説明書を付けたパッケージを配布しています。
GitHub 版でもほぼ同様のものは作成可能ですが、サポートなどは行いません。
